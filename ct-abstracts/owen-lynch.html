An Invitation to Geometric Type Theory

Owen Lynch

Joint work with: David Jaz Myers

Geometric Type Theory is a conjectured type theory that would provide an internal language for the 2-category of topoi [6]. In this talk, we give a direct construction of a category with familes (CwF, [1]) permitting interpretation of such a type theory. This is inspired by a close reading of the Elephant [3]; Johnstone defines a geometric theory over a fixed topos 𝒮 to be a 2-functor 𝒯 from (𝖡𝖳𝗈𝗉/𝒮)op to 𝖢𝖺𝗍 that is built by applying a sequence of PIE limits to the object classifier T0⁢(ℰ)=ℰ. From this, we define a CwF:

CwF sort Interpretation
Context Γ Geometric theory Γ:𝖳𝗈𝗉op→𝖢𝖺𝗍
Substitution Δ⊢γ:Γ 2-natural transformation γ:Δ⇒Γ
Type Γ⊢A⁢𝗍𝗒𝗉𝖾 Geometric theory over Γ, that is A:∫Γ→𝖢𝖺𝗍
Term Γ⊢a:A Natural section a⁢(ℰ,M)∈A⁢(ℰ,M) for (ℰ,M)∈∫Γ

Note that ∫Γ is equivalent to topoi sliced over the classifying topos for Γ, by representability of geometric theories. This CwF then supports a rich variety of type formers.

Example 1.

There is a type Γ⊢𝖲𝗈𝗋𝗍⁢𝗍𝗒𝗉𝖾 that is is given semantics by the object classifier ∫Γ→𝖢𝖺𝗍 defined by (ℰ,M)↦ℰ. Given Γ⊢A:𝖲𝗈𝗋𝗍, we have another type Γ⊢𝖤𝗅𝗍⁢A⁢𝗍𝗒𝗉𝖾 defined by (ℰ,M)↦ℰ⁢(1,A⁢(ℰ,M)), so 𝖲𝗈𝗋𝗍 acts as a “universe of small types”. The object classifier 𝖲𝗈𝗋𝗍 serves a dual role: it helps us build up theories, but additionally when we work internally to 𝖲𝗈𝗋𝗍 in a fixed context Γ we get the positive fragment of the internal language for the classifying topos for Γ.

Example 2.

There is a “Π-type with small codomain” (x:A)→B defined for Γ⊢A:𝖲𝗈𝗋𝗍 and Γ,x:𝖤𝗅𝗍⁢A⊢B⁢𝗍𝗒𝗉𝖾. The interpretation of (x:A)→B on (ℰ,M)∈∫Γ is given by interpreting B⁢[x] in the slice topos ℰ/A⁢(ℰ,M), where x is the global element of A in ℰ/A⁢(ℰ,M) given by the identity.

We also have Σ-types, both globally and within the 𝖲𝗈𝗋𝗍 universe.

In fact, this construction scales beyond just topoi/geometric theories. From Garner and Lack [2], we know that various fragments of geometric logic correspond to sub 2-monads of 𝖯𝗌𝗁:𝖫𝖾𝗑→𝖫𝖾𝗑 given by completion under various classes of weighted colimits. Type theoretically, this corresponds to allowing different kinds of (strictly positive) type formers. For instance, regular logic corresponds to adding propositional truncation, with the elimination principle as given in the HoTT book [4]. Especially interesting for us is the setting of arithmetic universes [5], which interprets quotient inductive inductive types.
