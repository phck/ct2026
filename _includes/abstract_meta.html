{%- comment -%} Talk-details "eyebrow" for an abstract page. Joins the page's `slug` (front matter) to the schedule data and renders weekday · date · time · room, plus a "ZOOM" marker for remote talks. Styling: .abstract-meta in assets/css/abstract.css. Lookup scope: CT reads _data/schedule.yml (invited = plenary_room; contributed = plenary_room; parallel/lightning rooms indexed positionally to schedule.rooms). DV60 reads _data/dv60-schedule.yml (single room: sched.room). Slugs are unique, so the last match wins harmlessly. Renders nothing if the slug isn't scheduled (e.g. cancelled talks, or DV60 abstracts still pending a schedule slot). {%- endcomment -%} {%- assign slug = page.slug -%} {%- assign found = false -%} {%- if page.conference == 'dv60' -%} {%- assign sched = site.data['dv60-schedule'] -%} {%- for day in sched.days -%} {%- for t in day.talks -%} {%- if t.slug == slug -%} {%- assign found = true -%} {%- assign m_day = day.date -%}{%- assign m_time = t.time -%} {%- assign m_room = sched.room -%}{%- assign m_mode = t.mode -%} {%- endif -%} {%- endfor -%} {%- endfor -%} {%- else -%} {%- assign sched = site.data.schedule -%} {%- for day in sched.days -%} {%- if day.invited and day.invited.slug == slug -%} {%- assign found = true -%} {%- assign m_day = day.date -%}{%- assign m_time = day.invited.time -%} {%- assign m_room = sched.plenary_room -%}{%- assign m_mode = day.invited.mode -%} {%- endif -%} {%- for t in day.contributed -%} {%- if t.slug == slug -%} {%- assign found = true -%} {%- assign m_day = day.date -%}{%- assign m_time = t.time -%} {%- assign m_room = sched.plenary_room -%}{%- assign m_mode = t.mode -%} {%- endif -%} {%- endfor -%} {%- if day.parallel -%} {%- for room in day.parallel.rooms -%} {%- assign ri = forloop.index0 -%} {%- for t in room -%} {%- if t.slug == slug -%} {%- assign found = true -%} {%- assign m_day = day.date -%}{%- assign m_time = t.time -%} {%- assign m_room = sched.rooms[ri] -%}{%- assign m_mode = t.mode -%} {%- endif -%} {%- endfor -%} {%- endfor -%} {%- endif -%} {%- if day.lightning -%} {%- for room in day.lightning.rooms -%} {%- assign ri = forloop.index0 -%} {%- for t in room -%} {%- if t.slug == slug -%} {%- assign found = true -%} {%- assign m_day = day.date -%}{%- assign m_time = t.time -%} {%- assign m_room = sched.rooms[ri] -%}{%- assign m_mode = t.mode -%} {%- endif -%} {%- endfor -%} {%- endfor -%} {%- endif -%} {%- endfor -%} {%- endif -%} {%- if found -%} {%- assign datestr = m_day | date: "%a · %b %-d" | upcase -%}
